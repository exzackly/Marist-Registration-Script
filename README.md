Marist Registration Script
==============
Marist sub-second automatic registration script

INSTALLATION - Using Greasemonkey on Firefox or Tampermonkey on Chrome, create a new script, then copy the script into it
 
USAGE – Enter your CRN numbers between the [] on line 15. Choose the time to start the sniper (typically 1 minute before registration) on line 28; follow the format described in line 27. Enable the script, and go to the Student tab in Banner Self Service, then click Add or Drop Classes. The script will execute at the selected time
