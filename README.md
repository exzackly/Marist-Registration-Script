Marist Registration Script
==============
Marist sub-second automatic registration script

INSTALLATION - Using Greasemonkey on Firefox or Tampermonkey on Chrome, create a new script, then copy the script into it
 
USAGE – Choose the time to start the sniper (typically 1 minute before registration) on line 13; follow the format described on line 12. Enter your CRNs (separated by a comma) between the [] on line 14. Enable the script, go to the Student tab in Banner Self Service, then click Add or Drop Classes. The script will execute at the selected time
